  <files>server/api/volumes.py</files>
  <action>
Add a new async endpoint `get_volume_as_nifti` at route `GET /{volume_id}/nifti`.

Implementation:
1. If `volume_id not in _metadata_registry`: raise HTTPException 404.
2. Call `_ensure_loaded(volume_id)` to populate cache.
3. Retrieve `data, loader_metadata` from `_volume_cache[volume_id]`.
4. Retrieve `path, fmt` from `_path_registry[volume_id]`.

For NIfTI source volumes (`fmt == "nifti"`):
- Read the raw file bytes directly from `Path(path)` — no conversion needed.
- Determine filename: `Path(path).name` (preserves .nii or .nii.gz extension).
- Return `Response(content=file_bytes, media_type="application/octet-stream", headers={"Content-Disposition": f'attachment; filename="{filename}"'})`.

For DICOM source volumes (`fmt in ("dicom", "dicom_series")`):
- Import nibabel at function scope: `import nibabel as nib`.
- Import io: `import io`.
- `meta = _metadata_registry[volume_id]`
- Build affine from voxel spacing: `affine = np.diag([*meta.voxel_spacing, 1.0])` (simple identity scaled by spacing; no RAS transform needed for external tool use).
- Create NIfTI image: `img = nib.Nifti1Image(data.astype(np.float32), affine)`.
- Serialize to bytes: `buf = io.BytesIO(); nib.save(img, buf)` — NOTE: nibabel's `nib.save` requires a filename to pick format. Instead use `file_map = img.make_file_map({'image': nib.FileHolder(fileobj=buf)}); img.to_file_map(file_map)` pattern, OR simpler: use `nib.Nifti1Image` and call `img.to_bytes()` if available, otherwise serialize via `nib.streamers` or write to a temp file.

  Simplest reliable approach for DICOM: use `tempfile.NamedTemporaryFile` with suffix `.nii`:
  ```python
  import tempfile, os
  with tempfile.NamedTemporaryFile(suffix=".nii", delete=False) as tmp:
      tmp_path = tmp.name
  try:
      nib.save(img, tmp_path)
      with open(tmp_path, "rb") as f:
          file_bytes = f.read()
  finally:
      os.unlink(tmp_path)
  ```
- Filename: `f"{meta.name}.nii"`.
- Return Response with file_bytes, octet-stream, Content-Disposition attachment.

Add import for `numpy as np` if not already present at top of file (check existing imports first).

Locate the existing `aiBtn` block (around line 695-702). Immediately after `toolPanel.appendChild(aiBtn);`, add:

1. A new `_runTotalSegmentator` async function (define it near `_showAIModelPicker` or inline as a named function before `_setupToolPanel`):

```js
async function _runTotalSegmentator(state, metadata) {
  if (!metadata || !metadata.id) {
    alert('No volume loaded.');
    return;
  }
  const confirmed = window.confirm(
    'This will download the volume as NIfTI and open TotalSegmentator.\n\n' +
    'Upload the downloaded file at totalsegmentator.com to run segmentation.'
  );
  if (!confirmed) return;

  // Trigger NIfTI download
  const url = `/api/v1/volumes/${metadata.id}/nifti`;
  const a = document.createElement('a');
  a.href = url;
  a.download = '';
  document.body.appendChild(a);
  a.click();
  document.body.removeChild(a);

  // Open TotalSegmentator in new tab
  window.open('https://totalsegmentator.com', '_blank', 'noopener,noreferrer');
}
```

2. Add the button after `toolPanel.appendChild(aiBtn);`:

```js
// TotalSegmentator button
const tsBtn = document.createElement('button');
tsBtn.className = 'compact-btn';
tsBtn.textContent = 'TotalSegmentator';
tsBtn.title = 'Download volume as NIfTI and open TotalSegmentator';
tsBtn.style.cssText = 'width:100%;margin-top:4px;padding:4px 8px;font-size:12px;';
tsBtn.addEventListener('click', () => _runTotalSegmentator(state, metadata));
toolPanel.appendChild(tsBtn);
```

Note: `_runTotalSegmentator` does NOT use fetch/blob to avoid memory overhead for large volumes — it uses a direct anchor href which lets the browser stream the download. This is intentional.
