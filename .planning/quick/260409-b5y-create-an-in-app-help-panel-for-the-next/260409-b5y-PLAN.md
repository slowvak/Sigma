helpButton.style.marginLeft = 'auto';

Update the return statement to include helpButton:

**main.js changes:**

1. Add import at top: `import { openHelpModal } from './ui/helpModal.js';`

2. In the shell setup (where prefsButton is wired), add:
```js
helpButton.addEventListener('click', openHelpModal);
```

3. Add global keydown listener for "?" (after shell is created, outside any volume-specific setup):
```js
document.addEventListener('keydown', (e) => {
  // Only trigger when no input/textarea/select is focused
  const tag = document.activeElement?.tagName;
  if (tag === 'INPUT' || tag === 'TEXTAREA' || tag === 'SELECT') return;
  if (e.key === '?') {
    e.preventDefault();
    openHelpModal();
  }
});
```
